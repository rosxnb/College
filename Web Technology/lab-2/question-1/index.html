Web Techonology

Course Title: Web Technology Full Marks: 60+20+20
Course No: CSC318 Pass Marks: 24+8+8
Nature of the Course: Theory + Lab Credit hours: 3
Semester: V

Course Description: This course covers the fundamental concepts of HTML, CSS, JavaScript, XML, and PHP.

Course Objectives: The objective of this course are:

  • Basic knowledge of web design
  • Client Side Scripting with JavaScript
  • Server Side Scripting with PHP
  • Handling web data using XML
Unit I: Introduction
  1. Web Basics: Internet, Intranet, WWW, Static and Dynamic Web Page; Web Clients; Web Servers
  2. Client Server Architecture: Single Tier, Two-Tier, Multi-Tier
  3. HTTP: HTTP Request and Response; URL
  4. Client Side Scripting, Server Side Scripting
  5. Web 1.0, Web 2.0
